# LinkedIn Posts - Quill AI v1.0.0

## Variant 1 - Story and vision

Today I am launching Quill AI v1.0.0.

I built Quill because most AI workflows are fragmented: one tool for chat, another for code, another for generation, and another for review.
Quill brings this into one loop:
ask -> generate artifact -> inspect in canvas -> refine.

What it does now:
- Personal AI agent for research, writing, and coding
- Builder targets for page, react-app, and nextjs-bundle
- Canvas-based code plus preview iteration loop
- Open source repository with contributor and security foundations

GitHub:
https://github.com/tovrr/quill-ai
