Title: Dependency audit and upgrade recommendations

## Purpose

Guide an agent to audit `package.json` dependencies for known vulnerabilities, outdated versions, and recommend safe upgrades.

## Inputs

- `packageFile`: path to `package.json` (default root).
- `policy`: `conservative` (minor/patch only) or `upgrade-all`.

## Tasks

1. Read `package.json` and identify direct deps/devDeps.
2. Query registry metadata (versions) and CVE advisories (if available).
3. Produce a report with upgrade suggestions and an optional `apply_patch` to bump versions in `package.json`.
4. Recommend follow-up steps (run `npm ci`, run `npm run typecheck`, run tests).

## Output (JSON)

{
"report": { "outdated": [...], "vulnerabilities": [...] },
"patch": "optional apply_patch or git commands",
"notes": "Run CI tests after upgrading"
}
