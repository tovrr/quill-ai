Title: Generate unit & integration tests

## Purpose

Guide an agent to scaffold deterministic unit and small integration tests for a given module or feature.

## Inputs

- `target`: file or module to test (e.g., `src/lib/prompt-sanitizer.ts`).
- `framework`: `vitest` (default) or `jest`.
- `coverageGoal`: optional percentage target.

## Tasks

1. Inspect exported functions and side effects.
2. For each public function, generate one or more focused unit tests (happy path, edge cases, error paths).
3. If integration tests are requested, produce a minimal test that runs the function in a near-real environment (e.g., `node --loader` or small mock of dependencies).
4. Output test files and any required test fixtures/mocks.

## Output (JSON)

{
"tests": [{ "path": "test/prompt-sanitizer.test.ts", "content": "..." }],
"run": "npx vitest --run",
"notes": "Add snapshots for complex outputs"
}
