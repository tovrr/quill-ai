Title: Refactor modules and reorganize code

## Purpose

Help an agent produce a safe, reviewable refactor plan for moving or renaming modules, extracting utilities, and adding small shims to preserve imports.

## Inputs

- `scope`: path or module name to refactor (e.g., `src/lib/chat`).
- `style`: `conservative` or `aggressive`.
- `goals`: list of refactor goals (e.g., "extract web-search integration", "reduce circular deps").

## Tasks

1. Analyze imports and identify dependent files.
2. Propose a minimal target layout and a sequence of `git mv` and shim files to preserve imports.
3. Produce an `apply_patch` diff or exact `git mv` commands for each step.
4. Suggest tests and typechecks to run after each step.

## Output (JSON)

{
"summary": "...",
"proposedMoves": [{ "from": "src/lib/x.ts", "to": "src/lib/new/x.ts", "commands": ["git mv ..."] }],
"shims": [{ "path": "src/lib/x.ts", "content": "export * from './new/x'" }],
"checklist": ["npm run typecheck", "npm run test:smoke"],
"rollback": ["git checkout -- <file>"]
}
