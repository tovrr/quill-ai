Title: Create smoke tests for critical flows

## Purpose

Guide an agent to create quick smoke tests that validate critical runtime flows (auth, simple API, builder end-to-end smoke).

## Inputs

- `flows`: array of flow identifiers (e.g., `auth-login`, `chat-builder-smoke`).
- `env`: environment to use (`dev`, `ci`).

## Tasks

1. For each flow, identify minimal setup steps and required env vars.
2. Produce small scripts or test files (e.g., `test/smoke/auth.test.ts`) that run the flow and assert basic expectations.
3. Recommend a lightweight runner (node script or vitest) and timeout values.

## Output (JSON)

{
"smokeTests": [{ "path":"test/smoke/auth.test.ts","content":"..." }],
"run": "npx vitest test/smoke --run"
}
